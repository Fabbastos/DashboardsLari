## 📈 Visão de Gestão & Valor Agregado

Este projeto utiliza o **Pandas** para criar uma "Single Source of Truth" (Fonte Única da Verdade), eliminando as inconsistências comuns de planilhas manuais.

* **Integridade Decisória:** Modelagem de dados rigorosa que garante KPIs auditáveis e livres de erros de digitação ou formatação.
* **Agilidade no Pipeline:** Redução drástica no tempo de preparação de dados (Data Wrangling), permitindo foco total na análise.
* **Mitigação de Risco:** Monitoramento preciso de saldos devedores e conversão de moedas, essencial para operações internacionais.

---

## 🧠 Engine de Dados & Tecnologia (DX)

A arquitetura foi desenhada para ser rápida e resiliente, utilizando o melhor do ecossistema Python:

* **Processamento com Pandas:** Otimização de grandes volumes de dados através de operações vetorizadas para limpeza e agregação de métricas.
* **Sanitização Inteligente:** Tratamento automático de moedas europeias (€), datas e caracteres especiais via Regex e tipos nativos do Pandas.
* **UI/UX Premium:** Experiência *Dark Mode* customizada com CSS para dashboards de alto nível executivo.

## 📁 Estrutura da Planilha (Data Schema)

O dashboard espera que o Pandas processe as seguintes colunas:

| Coluna | Tipo Esperado | Impacto no Dashboard |
| :--- | :--- | :--- |
| `Mês` | String/Date | Séries temporais e tendências mensais. |
| `Canal` | String | Segmentação de origem (Lead, Kalil, etc). |
| `Valor` | Currency/Float | Cálculo de receita e ROI. |
| `País` | String | Inteligência geográfica e expansão. |
